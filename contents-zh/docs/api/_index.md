## XChain API

Chain API 是运行在 `main api` 及 `newbridge core`后端的程序，其接口以 [grpc](https://grpc.io) 定义。

`ChainAPI` 服务定义如下：

```grpc
// The ChainAPI service definition.
service ChainAPI {
    rpc CreateAccount (CreateAccountRequest) returns (CreateAccountReply) {}
}
```
